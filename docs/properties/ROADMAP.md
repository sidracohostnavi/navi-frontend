# Properties Product Roadmap

## Phase 1: Core Management (Completed)
- [x] **CRUD:** Create, Read, Update, Delete properties.
- [x] **Import:** Basic scraping from URL.
- [x] **Images:** Display listing images.

## Phase 2: Enhanced Details (Current)
- [ ] **Amenities:** Structured list (Wifi, Pool, Parking) with icons.
- [ ] **Access Codes:** Field for Smart Lock codes or Keybox combo.
- [ ] **Guidebooks:** Digital House Manual linked to the property.

## Phase 3: Operations
- [ ] **Cleaners:** Assign a specific cleaner or cleaning company to a property.
- [ ] **Inventory:** Track supplies (Toilet paper, soap) per unit.
- [ ] **Maintenance:** Log tickets against specific properties.

## Phase 4: Multi-Unit
- [ ] **Parent/Child:** Support for "Room A inside House B".
- [ ] **Bulk Edit:** Apply settings (like Check-in time) to 50 properties at once.
